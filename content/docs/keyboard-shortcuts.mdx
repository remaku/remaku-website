| Shortcut     | Action                |
| ------------ | --------------------- |
| Ctrl+N       | New macro             |
| Ctrl+,       | Open settings         |
| Ctrl+Shift+N | Add step              |
| Delete       | Delete selected steps |
| Alt+Up       | Move step up          |
| Alt+Down     | Move step down        |
| Ctrl+Z       | Undo                  |
| Ctrl+Y       | Redo                  |
| Ctrl+D       | Duplicate steps       |
| Ctrl+C       | Copy steps            |
| Ctrl+X       | Cut steps             |
| Ctrl+V       | Paste steps           |
