title: 鍵盤快捷鍵
description: Remaku 常用快捷鍵。
icon: Keyboard
date: '2026-06-18T09:33:00Z'
---

| 快捷鍵       | 功能         |
| ------------ | ------------ |
| Ctrl+N       | 新增巨集     |
| Ctrl+,       | 開啟設定     |
| Ctrl+Shift+N | 新增步驟     |
| Delete       | 刪除所選步驟 |
| Alt+上       | 步驟上移     |
| Alt+下       | 步驟下移     |
| Ctrl+Z       | 復原         |
| Ctrl+Y       | 重做         |
| Ctrl+D       | 複製步驟     |
| Ctrl+C       | 複製步驟     |
| Ctrl+X       | 剪下步驟     |
| Ctrl+V       | 貼上步驟     |
