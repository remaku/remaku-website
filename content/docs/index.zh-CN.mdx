Remaku 是开源、可视化、以图像识别为核心的 Windows 桌面宏工具。你可以用截图、键盘输入、鼠标动作、等待、循环与分支创建自动化流程，不需要写代码。

## 下载

从 GitHub Releases 下载最新版安装程序：[下载最新版](https://github.com/remaku/remaku/releases/latest/download/Remaku_Setup.exe)

你也可以前往 [GitHub](https://github.com/remaku/remaku)。

## 安装注意事项

首次运行时，Windows SmartScreen 可能会显示"Windows 已保护你的电脑"警告，点击"更多信息"->"仍要运行"即可正常执行。

这个警告无害，原因是程序没有经过代码签名。本项目是开源软件，代码签名证书每年需要 $200 美元以上，因此目前没有签名。

## 基本流程

1. 创建宏。
2. 捕获或选择模板图片。
3. 添加按键、鼠标动作、等待图片、分支与循环等步骤。
4. 设置目标窗口与可选热键。
5. 运行宏，通过状态栏或浮窗查看进度。
