Remaku 是開源、視覺化、以圖像辨識為核心的 Windows 桌面巨集工具。你可以用截圖、鍵盤輸入、滑鼠動作、等待、迴圈與分支建立自動化流程，不需要寫程式。

## 下載

從 GitHub Releases 下載最新版安裝程式：[下載最新版](https://github.com/remaku/remaku/releases/latest/download/Remaku_Setup.exe)

你也可以前往 [GitHub](https://github.com/remaku/remaku)。

## 安裝注意事項

首次執行時，Windows SmartScreen 可能會顯示「Windows 已保護您的電腦」警告，點擊「其他資訊」→「仍要執行」即可正常執行。

這個警告無害，原因是程式沒有經過程式碼簽章。本專案是開源軟體，程式碼簽章憑證每年需要 $200 美元以上，因此目前沒有簽章。

## 基本流程

1. 建立巨集。
2. 擷取或選擇模板圖片。
3. 新增按鍵、滑鼠動作、等待圖片、分支與迴圈等步驟。
4. 設定目標視窗與可選的熱鍵。
5. 執行巨集，透過狀態列或浮窗查看進度。
