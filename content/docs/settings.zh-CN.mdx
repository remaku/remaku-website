## 宏

以下设置针对各个宏，在右侧面板进行调整。

| 项目     | 说明                                               |
| -------- | -------------------------------------------------- |
| 目标窗口 | 运行宏前 Remaku 要查找的窗口标题。                 |
| 热键     | 此宏专用的全局快捷键。                             |
| 游戏模式 | 根据游戏分辨率变化自动缩放模板。桌面程序建议关闭。 |
| 后台输入 | 不切换焦点，直接将输入发送到目标窗口。             |
| 防止失焦 | 防止目标窗口检测到失焦，切换窗口时游戏不会暂停。   |

## 常规

| 项目            | 说明                                     |
| --------------- | ---------------------------------------- |
| 窗口置顶        | 让 Remaku 保持在其他窗口上方。           |
| 显示状态浮窗    | 显示宏执行状态的透明浮动窗口。           |
| 启动时检查更新  | 启动 Remaku 时自动检查新版本。           |
| 更新频道        | 选择稳定版或测试版。                     |
| 主题            | 跟随系统、浅色或深色。                   |
| 语言            | 自动检测、繁体中文、简体中文或 English。 |
| 暂停/继续快捷键 | 暂停或继续当前宏的全局快捷键。           |

## 捕获

| 项目 | 说明           |
| ---- | -------------- |
| FPS  | 每秒捕获帧数。 |

## 输入

| 项目      | 说明                         |
| --------- | ---------------------------- |
| 抖动 (ms) | 每次按键加入的随机延迟范围。 |
