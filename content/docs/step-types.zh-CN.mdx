| 步骤             | 说明                                                 |
| ---------------- | ---------------------------------------------------- |
| 按键             | 按下键盘按键或修饰键组合。                           |
| 延迟             | 等待固定的毫秒数。                                   |
| 文本输入         | 输入文本，可设定每个字符之间的延迟。                 |
| 鼠标点击         | 在屏幕坐标或找到的图片中心点击。                     |
| 鼠标移动         | 将光标移到坐标或找到的图片中心。                     |
| 鼠标滚轮         | 滚动鼠标滚轮指定格数。                               |
| 等待图片         | 暂停直到模板图片出现在屏幕上。                       |
| 等待数字         | 暂停直到捕获区域中的数字达到目标值。                 |
| 长按直到图片消失 | 图片可见时按住按键，图片消失时松开。                 |
| 重复             | 将子步骤重复执行指定次数。                           |
| 重复直到数字     | 重复执行子步骤直到区域中的数字达到目标值。           |
| 如果图片         | 分支：图片找到时执行 then 步骤，否则执行 else 步骤。 |
| 如果数字         | 根据捕获区域中的数字进行条件分支。                   |
| 如果任一图片     | 同时监控多个模板，执行第一个匹配项的对应分支。       |
| 网格导航         | 逐格轮替操作，可设定每行/每列的自定义动作。          |

每个步骤都有**跳过**开关可暂时停用，以及可选的**备注**栏位用来记录说明。

---

### 按键

![按键步骤属性](/images/docs/step-types/key.zh-CN.png)

| 字段          | 默认值  | 说明                       |
| ------------- | ------- | -------------------------- |
| 按键          | `enter` | 要按下的按键或修饰键组合。 |
| 按住时间 (ms) | `90`    | 按键按住的持续时间。       |

---

### 延迟

![延迟步骤属性](/images/docs/step-types/delay.zh-CN.png)

| 字段          | 默认值 | 说明           |
| ------------- | ------ | -------------- |
| 持续时间 (ms) | `500`  | 等待的毫秒数。 |

---

### 文本输入

![文本输入步骤属性](/images/docs/step-types/text-input.zh-CN.png)

| 字段      | 默认值   | 说明                                 |
| --------- | -------- | ------------------------------------ |
| 文本      | _(空白)_ | 要输入的文本，支持 Unicode。         |
| 间隔 (ms) | `0`      | 每个字符之间的延迟（0 = 最快速度）。 |

---

### 鼠标点击

![鼠标点击步骤属性](/images/docs/step-types/mouse-click.zh-CN.png)

**目标为坐标时：**

| 字段               | 默认值   | 说明                                           |
| ------------------ | -------- | ---------------------------------------------- |
| 按钮               | `左键`   | 鼠标按钮：左键、右键或中键。                   |
| 目标               | `坐标`   | 在屏幕坐标点击。                               |
| X                  | `0`      | 水平位置（像素）。                             |
| Y                  | `0`      | 垂直位置（像素）。                             |
| 相对               | `客户区` | `客户区` = 相对于目标窗口，`绝对` = 屏幕坐标。 |
| 按下/松开延迟 (ms) | `50`     | 松开按钮前的按住时间。                         |

**目标为图片中心时：**

| 字段               | 默认值     | 说明                         |
| ------------------ | ---------- | ---------------------------- |
| 按钮               | `左键`     | 鼠标按钮：左键、右键或中键。 |
| 目标               | `图片中心` | 在找到的图片中心点击。       |
| 模板               | _(无)_     | 要查找的模板图片。           |
| 相似度             | `85%`      | 所需的匹配信心度。           |
| 超时 (ms)          | `5000`     | 等待图片出现的时间。         |
| 超时动作           | `停止`     | `停止`宏或`继续`执行下一步。 |
| 按下/松开延迟 (ms) | `50`       | 松开按钮前的按住时间。       |

---

### 鼠标移动

![鼠标移动步骤属性](/images/docs/step-types/mouse-move.zh-CN.png)

**目标为坐标时：**

| 字段 | 默认值   | 说明                                           |
| ---- | -------- | ---------------------------------------------- |
| 目标 | `坐标`   | 移动到屏幕坐标。                               |
| X    | `0`      | 水平位置（像素）。                             |
| Y    | `0`      | 垂直位置（像素）。                             |
| 相对 | `客户区` | `客户区` = 相对于目标窗口，`绝对` = 屏幕坐标。 |

**目标为图片中心时：**

| 字段      | 默认值     | 说明                         |
| --------- | ---------- | ---------------------------- |
| 目标      | `图片中心` | 移动到找到的图片中心。       |
| 模板      | _(无)_     | 要查找的模板图片。           |
| 相似度    | `85%`      | 所需的匹配信心度。           |
| 超时 (ms) | `5000`     | 等待图片出现的时间。         |
| 超时动作  | `停止`     | `停止`宏或`继续`执行下一步。 |

### 鼠标滚轮

![鼠标滚轮步骤属性](/images/docs/step-types/mouse-scroll.zh-CN.png)

| 字段      | 默认值 | 说明                                   |
| --------- | ------ | -------------------------------------- |
| 滚轮格数  | `3`    | 滚动刻度数。正值 = 向上，负值 = 向下。 |
| 间隔 (ms) | `0`    | 每个滚轮刻度之间的延迟。               |

### 等待图片

![等待图片步骤属性](/images/docs/step-types/wait-image.zh-CN.png)

| 字段      | 默认值 | 说明                         |
| --------- | ------ | ---------------------------- |
| 模板      | _(无)_ | 要等待出现的模板图片。       |
| 相似度    | `85%`  | 所需的匹配信心度（0–100%）。 |
| 超时 (ms) | `5000` | 放弃前的等待时间。           |
| 超时动作  | `停止` | `停止`宏或`继续`执行下一步。 |

---

### 等待数字

![等待数字步骤属性](/images/docs/step-types/wait-number.zh-CN.png)

| 字段         | 默认值   | 说明                                           |
| ------------ | -------- | ---------------------------------------------- |
| X            | `0`      | 捕获区域的水平位置。                           |
| Y            | `0`      | 捕获区域的垂直位置。                           |
| 宽度         | `100`    | 捕获区域的宽度（像素）。                       |
| 高度         | `30`     | 捕获区域的高度（像素）。                       |
| 捕获宽度     | _(自动)_ | 捕获时使用的屏幕分辨率宽度。                   |
| 捕获高度     | _(自动)_ | 捕获时使用的屏幕分辨率高度。                   |
| 相对         | `客户区` | `客户区` = 相对于目标窗口，`绝对` = 屏幕坐标。 |
| 比较方式     | `≥`      | 比较方式：`=`、`≠`、`>`、`≥`、`<`、`≤`。       |
| 数值         | `0`      | 用来比较的目标数字。                           |
| 超时 (ms)    | `5000`   | 放弃前的等待时间。                             |
| 稳定读取次数 | `2`      | 数值需稳定读取几次才算有效。                   |

使用**选取区域**按钮选择包含数字的屏幕区域。Remaku 会以光学方式读取数字，等待条件满足后继续。

---

### 长按直到图片消失

![长按直到图片消失步骤属性](/images/docs/step-types/hold-key-until-gone.zh-CN.png)

| 字段          | 默认值   | 说明                           |
| ------------- | -------- | ------------------------------ |
| 按键          | `enter`  | 图片可见时要按住的按键。       |
| 模板          | _(无)_   | 要监视的模板图片。             |
| 相似度        | `85%`    | 所需的匹配信心度。             |
| 加载延迟 (ms) | `2000`   | 宏启动后开始检查前的缓冲时间。 |
| 查找超时 (ms) | `15000`  | 等待图片首次出现的时间。       |
| 消失宽限 (ms) | `1500`   | 图片需消失多久才松开按键。     |
| 绝对超时 (ms) | `180000` | 最大总执行时间，超出则停止宏。 |

### 重复

![重复步骤属性](/images/docs/step-types/repeat.zh-CN.png)

| 字段 | 默认值 | 说明                   |
| ---- | ------ | ---------------------- |
| 次数 | `1`    | 重复执行子步骤的次数。 |

在步骤树中将子步骤加入到 Repeat 区块内。

### 重复直到数字

![重复直到数字步骤属性](/images/docs/step-types/repeat-until-number.zh-CN.png)

| 字段         | 默认值   | 说明                                           |
| ------------ | -------- | ---------------------------------------------- |
| X            | `0`      | 捕获区域的水平位置。                           |
| Y            | `0`      | 捕获区域的垂直位置。                           |
| 宽度         | `100`    | 捕获区域的宽度（像素）。                       |
| 高度         | `30`     | 捕获区域的高度（像素）。                       |
| 捕获宽度     | _(自动)_ | 捕获时使用的屏幕分辨率宽度。                   |
| 捕获高度     | _(自动)_ | 捕获时使用的屏幕分辨率高度。                   |
| 相对         | `客户区` | `客户区` = 相对于目标窗口，`绝对` = 屏幕坐标。 |
| 比较方式     | `≥`      | 比较方式：`=`、`≠`、`>`、`≥`、`<`、`≤`。       |
| 数值         | `0`      | 用来比较的目标数字。                           |
| 超时 (ms)    | `5000`   | 每次迭代等待稳定读取的时间。                   |
| 稳定读取次数 | `2`      | 数值需稳定读取几次才算有效。                   |
| 最大次数     | `1`      | 最大迭代次数（安全限制）。                     |
| 执行前先检查 | `开启`   | 是否在第一次迭代前先评估条件。                 |

将子步骤加入到 Repeat Until Number 区块中。每次迭代会检查条件，符合时结束循环。

### 如果图片

![如果图片步骤属性](/images/docs/step-types/if-image.zh-CN.png)

| 字段      | 默认值 | 说明                 |
| --------- | ------ | -------------------- |
| 模板      | _(无)_ | 要检查的模板图片。   |
| 相似度    | `85%`  | 所需的匹配信心度。   |
| 超时 (ms) | `5000` | 等待图片出现的时间。 |

将子步骤加入到 **Then** 分支（图片找到）与 **Else** 分支（图片未找到）。

### 如果任一图片

![如果任一图片步骤属性](/images/docs/step-types/if-any-image.zh-CN.png)

| 字段      | 默认值 | 说明                   |
| --------- | ------ | ---------------------- |
| 模板      | _(无)_ | 要扫描的多个模板图片。 |
| 相似度    | `85%`  | 所需的匹配信心度。     |
| 超时 (ms) | `5000` | 放弃前的等待时间。     |
| 超时动作  | `停止` | `停止`宏或`继续`。     |

将子步骤加入到每个模板的分支中。第一个匹配到的模板决定执行哪个分支。

### 网格导航

![网格导航步骤属性](/images/docs/step-types/grid-nav.zh-CN.png)

| 字段       | 默认值 | 说明                            |
| ---------- | ------ | ------------------------------- |
| 行数       | `1`    | 网格的行数。                    |
| 起始单元格 | `0`    | 从哪个单元格开始（从 0 起算）。 |

将子步骤加入到 **换行时** 与 **换列时** 分支，定义在网格中移动时的动作。
