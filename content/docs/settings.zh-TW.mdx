## 巨集

以下設定針對個別巨集，在右側面板進行調整。

| 項目     | 說明                                             |
| -------- | ------------------------------------------------ |
| 目標視窗 | 執行巨集前 Remaku 要尋找的視窗標題。             |
| 熱鍵     | 此巨集專用的全域快捷鍵。                         |
| 遊戲模式 | 依遊戲解析度變化自動縮放模板。桌面程式建議關閉。 |
| 背景輸入 | 不切換焦點，直接將輸入送到目標視窗。             |
| 防止失焦 | 防止目標視窗偵測到失焦，切換視窗時遊戲不會暫停。 |

## 一般

| 項目            | 說明                                     |
| --------------- | ---------------------------------------- |
| 視窗置頂        | 讓 Remaku 保持在其他視窗上方。           |
| 顯示狀態浮窗    | 顯示巨集執行狀態的透明浮動視窗。         |
| 啟動時檢查更新  | 啟動 Remaku 時自動檢查新版本。           |
| 更新頻道        | 選擇穩定版或測試版。                     |
| 主題            | 跟隨系統、淺色或深色。                   |
| 語言            | 自動偵測、繁體中文、簡體中文或 English。 |
| 暫停/繼續快捷鍵 | 暫停或繼續目前巨集的全域快捷鍵。         |

## 擷取

| 項目 | 說明           |
| ---- | -------------- |
| FPS  | 每秒擷取幀數。 |

## 輸入

| 項目      | 說明                         |
| --------- | ---------------------------- |
| 抖動 (ms) | 每次按鍵加入的隨機延遲範圍。 |
