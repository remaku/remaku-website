| 快捷键       | 功能         |
| ------------ | ------------ |
| Ctrl+N       | 新建宏       |
| Ctrl+,       | 打开设置     |
| Ctrl+Shift+N | 添加步骤     |
| Delete       | 删除所选步骤 |
| Alt+上       | 步骤上移     |
| Alt+下       | 步骤下移     |
| Ctrl+Z       | 撤销         |
| Ctrl+Y       | 重做         |
| Ctrl+D       | 复制步骤     |
| Ctrl+C       | 复制步骤     |
| Ctrl+X       | 剪切步骤     |
| Ctrl+V       | 粘贴步骤     |
