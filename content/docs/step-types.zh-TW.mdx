| 步驟             | 說明                                                 |
| ---------------- | ---------------------------------------------------- |
| 按鍵             | 按下鍵盤按鍵或修飾鍵組合。                           |
| 延遲             | 等待固定的毫秒數。                                   |
| 文字輸入         | 輸入文字，可設定每個字元之間的延遲。                 |
| 滑鼠點擊         | 在螢幕座標或找到的圖片中心點擊。                     |
| 滑鼠移動         | 將游標移到座標或找到的圖片中心。                     |
| 滑鼠滾輪         | 滾動滑鼠滾輪指定格數。                               |
| 等待圖片         | 暫停直到模板圖片出現在螢幕上。                       |
| 等待數字         | 暫停直到擷取區域中的數字達到目標值。                 |
| 長按直到圖片消失 | 圖片可見時按住按鍵，圖片消失時放開。                 |
| 重複             | 將子步驟重複執行指定次數。                           |
| 重複直到數字     | 重複執行子步驟直到區域中的數字達到目標值。           |
| 如果圖片         | 分支：圖片找到時執行 then 步驟，否則執行 else 步驟。 |
| 如果數字         | 根據擷取區域中的數字進行條件分支。                   |
| 如果任一圖片     | 同時監控多個模板，執行第一個匹配項的對應分支。       |
| 網格導航         | 逐格輪替操作，可設定每行/每列的自訂動作。            |

每個步驟都有**跳過**開關可暫時停用，以及可選的**備註**欄位用來記錄說明。

### 按鍵

![按鍵步驟屬性](/images/docs/step-types/key.zh-TW.png)

| 欄位          | 預設值  | 說明                       |
| ------------- | ------- | -------------------------- |
| 按鍵          | `enter` | 要按下的按鍵或修飾鍵組合。 |
| 按住時間 (ms) | `90`    | 按鍵按住的持續時間。       |

---

### 延遲

![延遲步驟屬性](/images/docs/step-types/delay.zh-TW.png)

| 欄位          | 預設值 | 說明           |
| ------------- | ------ | -------------- |
| 持續時間 (ms) | `500`  | 等待的毫秒數。 |

---

### 文字輸入

![文字輸入步驟屬性](/images/docs/step-types/text-input.zh-TW.png)

| 欄位      | 預設值   | 說明                                 |
| --------- | -------- | ------------------------------------ |
| 文字      | _(空白)_ | 要輸入的文字，支援 Unicode。         |
| 間隔 (ms) | `0`      | 每個字元之間的延遲（0 = 最快速度）。 |

---

### 滑鼠點擊

![滑鼠點擊步驟屬性](/images/docs/step-types/mouse-click.zh-TW.png)

**目標為座標時：**

| 欄位               | 預設值   | 說明                                           |
| ------------------ | -------- | ---------------------------------------------- |
| 按鈕               | `左鍵`   | 滑鼠按鈕：左鍵、右鍵或中鍵。                   |
| 目標               | `座標`   | 在螢幕座標點擊。                               |
| X                  | `0`      | 水平位置（像素）。                             |
| Y                  | `0`      | 垂直位置（像素）。                             |
| 相對               | `客戶區` | `客戶區` = 相對於目標視窗，`絕對` = 螢幕座標。 |
| 按下/放開延遲 (ms) | `50`     | 放開按鈕前的按住時間。                         |

**目標為圖片中心時：**

| 欄位               | 預設值     | 說明                           |
| ------------------ | ---------- | ------------------------------ |
| 按鈕               | `左鍵`     | 滑鼠按鈕：左鍵、右鍵或中鍵。   |
| 目標               | `圖片中心` | 在找到的圖片中心點擊。         |
| 模板               | _(無)_     | 要尋找的模板圖片。             |
| 相似度             | `85%`      | 所需的匹配信心度。             |
| 逾時 (ms)          | `5000`     | 等待圖片出現的時間。           |
| 逾時動作           | `停止`     | `停止`巨集或`繼續`執行下一步。 |
| 按下/放開延遲 (ms) | `50`       | 放開按鈕前的按住時間。         |

---

### 滑鼠移動

![滑鼠移動步驟屬性](/images/docs/step-types/mouse-move.zh-TW.png)

**目標為座標時：**

| 欄位 | 預設值   | 說明                                           |
| ---- | -------- | ---------------------------------------------- |
| 目標 | `座標`   | 移動到螢幕座標。                               |
| X    | `0`      | 水平位置（像素）。                             |
| Y    | `0`      | 垂直位置（像素）。                             |
| 相對 | `客戶區` | `客戶區` = 相對於目標視窗，`絕對` = 螢幕座標。 |

**目標為圖片中心時：**

| 欄位      | 預設值     | 說明                           |
| --------- | ---------- | ------------------------------ |
| 目標      | `圖片中心` | 移動到找到的圖片中心。         |
| 模板      | _(無)_     | 要尋找的模板圖片。             |
| 相似度    | `85%`      | 所需的匹配信心度。             |
| 逾時 (ms) | `5000`     | 等待圖片出現的時間。           |
| 逾時動作  | `停止`     | `停止`巨集或`繼續`執行下一步。 |

---

### 滑鼠滾輪

![滑鼠滾輪步驟屬性](/images/docs/step-types/mouse-scroll.zh-TW.png)

| 欄位      | 預設值 | 說明                                   |
| --------- | ------ | -------------------------------------- |
| 滾輪格數  | `3`    | 滾動刻度數。正值 = 向上，負值 = 向下。 |
| 間隔 (ms) | `0`    | 每個滾輪刻度之間的延遲。               |

---

### 等待圖片

![等待圖片步驟屬性](/images/docs/step-types/wait-image.zh-TW.png)

| 欄位      | 預設值 | 說明                           |
| --------- | ------ | ------------------------------ |
| 模板      | _(無)_ | 要等待出現的模板圖片。         |
| 相似度    | `85%`  | 所需的匹配信心度（0–100%）。   |
| 逾時 (ms) | `5000` | 放棄前的等待時間。             |
| 逾時動作  | `停止` | `停止`巨集或`繼續`執行下一步。 |

---

### 等待數字

![等待數字步驟屬性](/images/docs/step-types/wait-number.zh-TW.png)

| 欄位         | 預設值   | 說明                                           |
| ------------ | -------- | ---------------------------------------------- |
| X            | `0`      | 擷取區域的水平位置。                           |
| Y            | `0`      | 擷取區域的垂直位置。                           |
| 寬度         | `100`    | 擷取區域的寬度（像素）。                       |
| 高度         | `30`     | 擷取區域的高度（像素）。                       |
| 擷取寬度     | _(自動)_ | 擷取時使用的螢幕解析度寬度。                   |
| 擷取高度     | _(自動)_ | 擷取時使用的螢幕解析度高度。                   |
| 相對         | `客戶區` | `客戶區` = 相對於目標視窗，`絕對` = 螢幕座標。 |
| 比較方式     | `≥`      | 比較方式：`=`、`≠`、`>`、`≥`、`<`、`≤`。       |
| 數值         | `0`      | 用來比較的目標數字。                           |
| 逾時 (ms)    | `5000`   | 放棄前的等待時間。                             |
| 穩定讀取次數 | `2`      | 數值需穩定讀取幾次才算有效。                   |

使用**選取區域**按鈕選擇包含數字的螢幕區域。Remaku 會以光學方式讀取數字，等待條件滿足後繼續。

---

### 長按直到圖片消失

![長按直到圖片消失步驟屬性](/images/docs/step-types/hold-key-until-gone.zh-TW.png)

| 欄位          | 預設值   | 說明                             |
| ------------- | -------- | -------------------------------- |
| 按鍵          | `enter`  | 圖片可見時要按住的按鍵。         |
| 模板          | _(無)_   | 要監看的模板圖片。               |
| 相似度        | `85%`    | 所需的匹配信心度。               |
| 載入延遲 (ms) | `2000`   | 巨集啟動後開始檢查前的緩衝時間。 |
| 尋找逾時 (ms) | `15000`  | 等待圖片首次出現的時間。         |
| 消失寬限 (ms) | `1500`   | 圖片需消失多久才放開按鍵。       |
| 絕對逾時 (ms) | `180000` | 最大總執行時間，超過則停止巨集。 |

### 重複

![重複步驟屬性](/images/docs/step-types/repeat.zh-TW.png)

| 欄位 | 預設值 | 說明                   |
| ---- | ------ | ---------------------- |
| 次數 | `1`    | 重複執行子步驟的次數。 |

在步驟樹中將子步驟加入到 Repeat 區塊內。

### 重複直到數字

![重複直到數字步驟屬性](/images/docs/step-types/repeat-until-number.zh-TW.png)

| 欄位         | 預設值   | 說明                                           |
| ------------ | -------- | ---------------------------------------------- |
| X            | `0`      | 擷取區域的水平位置。                           |
| Y            | `0`      | 擷取區域的垂直位置。                           |
| 寬度         | `100`    | 擷取區域的寬度（像素）。                       |
| 高度         | `30`     | 擷取區域的高度（像素）。                       |
| 擷取寬度     | _(自動)_ | 擷取時使用的螢幕解析度寬度。                   |
| 擷取高度     | _(自動)_ | 擷取時使用的螢幕解析度高度。                   |
| 相對         | `客戶區` | `客戶區` = 相對於目標視窗，`絕對` = 螢幕座標。 |
| 比較方式     | `≥`      | 比較方式：`=`、`≠`、`>`、`≥`、`<`、`≤`。       |
| 數值         | `0`      | 用來比較的目標數字。                           |
| 逾時 (ms)    | `5000`   | 每次迭代等待穩定讀取的時間。                   |
| 穩定讀取次數 | `2`      | 數值需穩定讀取幾次才算有效。                   |
| 最大次數     | `1`      | 最大迭代次數（安全限制）。                     |
| 執行前先檢查 | `開啟`   | 是否在第一次迭代前先評估條件。                 |

將子步驟加入到 Repeat Until Number 區塊中。每次迭代會檢查條件，符合時結束迴圈。

![如果圖片步驟屬性](/images/docs/step-types/if-image.zh-TW.png)

| 欄位      | 預設值 | 說明                 |
| --------- | ------ | -------------------- |

### 如果任一圖片

![如果任一圖片步驟屬性](/images/docs/step-types/if-any-image.zh-TW.png)

| 欄位      | 預設值 | 說明                   |
| --------- | ------ | ---------------------- |
| 模板      | _(無)_ | 要掃描的多個模板圖片。 |
| 相似度    | `85%`  | 所需的匹配信心度。     |
| 逾時 (ms) | `5000` | 放棄前的等待時間。     |
| 逾時動作  | `停止` | `停止`巨集或`繼續`。   |

將子步驟加入到每個模板的分支中。第一個匹配到的模板決定執行哪個分支。

### 網格導航

![網格導航步驟屬性](/images/docs/step-types/grid-nav.zh-TW.png)

| 欄位       | 預設值 | 說明                            |
| ---------- | ------ | ------------------------------- |
| 行數       | `1`    | 網格的行數。                    |
| 起始儲存格 | `0`    | 從哪個儲存格開始（從 0 起算）。 |

將子步驟加入到 **換行時** 與 **換列時** 分支，定義在網格中移動時的動作。
